Timestamp microservice

Usage:

Providing a timestamp in either the unix format (ms since Jan 01 1970) or a natural string this API returns an Object in JSON notation containing both formats.

Example Usage:

https://fcc-timestampms.glitch.me/1514073600000
https://fcc-timestampms.glitch.me/24 December 2017
https://fcc-timestampms.glitch.me/Sun Dec 24 2017

Example Output:

{"unix":1514073600000,"natural":"Sun Dec 24 2017"}
